# 人机协作

## 概述

人机协作（Human-in-the-loop，HITL）是一种将人类判断融入 AI 决策过程的模式，在关键步骤需要人类确认或介入，确保 AI 系统的可靠性和安全性。

## 为什么重要

- **质量保证**：人类审核确保输出质量
- **风险控制**：关键决策需要人工确认
- **持续学习**：人类反馈帮助模型改进
- **合规要求**：某些场景必须有人类参与

## 介入时机

- **任务开始**：人类审核任务目标和范围
- **执行过程中**：关键步骤的实时监控
- **任务完成**：结果审核和确认
- **异常处理**：AI 无法处理的边界情况

## 与其他概念的关系

- 与 [Agent](/glossary/agent) 配合实现受控自动化
- 区别于完全 [自主 Agent](/glossary/autonomous-agent)
- 是 [AI 安全](/glossary/ai-safety) 的重要保障
