# 向量数据库

## 概述

向量数据库是专门用于存储和检索高维向量数据的数据库系统，支持高效的相似性搜索，是 [RAG](/glossary/rag) 系统的核心基础设施。

## 为什么重要

- **语义检索**：支持基于语义相似度的搜索
- **高效查询**：处理亿级向量的毫秒级检索
- **可扩展性**：支持大规模数据存储
- **AI 原生**：专为 AI 应用设计

## 核心特性

- **向量索引**：HNSW、IVF 等高效索引算法
- **相似度度量**：支持余弦、欧氏等多种度量
- **过滤支持**：支持带条件的向量搜索
- **多模态**：支持图像、音频等多模态向量

## 主流产品

- **Pinecone**：托管向量数据库
- **Weaviate**：开源向量搜索引擎
- **Milvus**：开源向量数据库
- **Chroma**：轻量级向量库
