# 联邦学习

## 概述

联邦学习是一种分布式机器学习技术，多个客户端在不共享原始数据的情况下协作训练模型，保护数据隐私。

## 为什么重要

- **隐私保护**：数据不出本地
- **合规要求**：满足数据保护法规
- **数据孤岛**：打破数据孤岛
- **边缘智能**：端侧模型训练

## 核心流程

1. 中央服务器下发初始模型
2. 各客户端使用本地数据训练
3. 客户端上传模型更新（不是数据）
4. 中央服务器聚合更新
5. 更新全局模型，重复步骤 1-4

## 技术特点

- **隐私保护**：原始数据不离开本地
- **通信效率**：只传输模型参数
- **异构数据**：处理非独立同分布数据
- **系统异构**：支持不同设备

## 应用场景

- **移动端**：手机键盘预测
- **医疗**：跨医院模型训练
- **金融**：跨机构风控模型
- **物联网**：边缘设备协作
