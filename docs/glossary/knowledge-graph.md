# 知识图谱

## 概述

知识图谱（Knowledge Graph）是以图结构表示实体及其关系的结构化知识表示方式，通常包含实体、属性和关系三种元素。

## 为什么重要

- **结构化知识**：将知识组织为可推理的结构
- **关系推理**：支持实体间关系的推理查询
- **可解释性**：知识来源和推理过程可追溯
- **知识整合**：整合多源异构数据

## 核心概念

- **实体**：现实世界的对象
- **属性**：实体的特征
- **关系**：实体之间的联系
- **三元组**：(头实体, 关系, 尾实体)

## 构建方法

- **自动化抽取**：从文本中自动抽取
- **众包编辑**：人工构建和维护
- **半自动构建**：自动+人工结合
- **知识融合**：多图谱合并

## 与其他概念的关系

- 与 [向量数据库](/glossary/vector-database) 互补
- 可作为 [RAG](/glossary/rag) 的知识源
- 支撑智能问答和推理
