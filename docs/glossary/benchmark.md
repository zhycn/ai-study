# 基准测试

## 概述

基准测试（Benchmark）是用于评估 AI 模型能力的标准化测试集，通过在统一数据集上测试，可以客观比较不同模型的性能。

## 为什么重要

- **性能评估**：客观量化模型能力
- **模型选择**：帮助选择合适的模型
- **迭代优化**：指引模型改进方向
- **领域标准**：建立行业共识

## 常见基准

- **MMLU**：多任务语言理解
- **HumanEval**：代码生成能力
- **MATH**：数学推理能力
- **BBH**：Big-Bench Hard 综合能力
- **CMMLU**：中文多任务理解

## 评估维度

- **知识**：事实性知识储备
- **推理**：逻辑和数学推理能力
- **编程**：代码理解和生成能力
- **创造力**：创意和写作能力
- **安全性**：对抗有害输入的能力
