# 记忆

## 概述

记忆（Memory）是指 AI Agent 存储和检索信息的能力，包括短期记忆（当前会话上下文）和长期记忆（跨会话积累的知识和经验）。

## 为什么重要

- **上下文连续性**：保持对话和任务的连续性
- **知识积累**：从历史经验中学习和改进
- **个性化**：根据用户偏好调整行为
- **任务延续**：支持长时间运行的多步骤任务

## 记忆类型

- **短期记忆**：当前会话的上下文信息
- **长期记忆**：持久化的知识和经验
- **工作记忆**：当前任务需要的临时信息
- **情景记忆**：特定场景的经历和结果

## 与其他概念的关系

- 支撑 [Agent](/glossary/agent) 的连续对话能力
- 与 [上下文窗口](/glossary/context-window) 相关但不同
- 是实现 [自主 Agent](/glossary/autonomous-agent) 的基础
