# 微调

## 概述

微调（Fine-tuning）是在预训练模型基础上，使用特定领域或任务的数据进行进一步训练，以适应特定需求的技术。

## 为什么重要

- **任务适配**：让通用模型适应特定任务
- **数据效率**：相比从头训练所需数据少
- **成本可控**：训练成本相对较低
- **效果显著**：可大幅提升特定任务效果

## 微调类型

- **全参数微调**：微调全部参数
- **LoRA**：低秩适配，微调少量参数
- **QLoRA**：量化+LoRA，更高效的微调
- **Prefix/Prompt Tuning**：调整提示而非模型

## 典型流程

1. 选择基础模型
2. 准备微调数据集
3. 配置训练参数
4. 执行微调训练
5. 评估微调效果
6. 部署微调模型

## 适用场景

- **领域适应**：医疗、法律等专业领域
- **指令跟随**：提升指令遵循能力
- **风格调整**：调整输出风格
- **特定任务**：特定类型任务优化
