# API

## 概述

API（Application Programming Interface，应用程序编程接口）是不同软件系统之间进行数据交换和功能调用的接口规范，是 AI 服务对外提供能力的标准方式。

## 为什么重要

- **服务化**：将 AI 能力封装为服务
- **标准化**：统一的调用方式
- **可扩展**：方便集成和扩展
- **计费基础**：API 调用是常见计费方式

## AI API 类型

- **模型 API**：调用大模型能力
- **Embedding API**：获取文本向量
- **图像 API**：图像生成、识别等
- **语音 API**：语音合成、识别

## 常见 AI API 服务

- **OpenAI API**：GPT 系列模型
- **Anthropic API**：Claude 系列模型
- **Google AI API**：Gemini 系列模型
- **国产大模型**：百度、阿里等

- 调用计费涉及 [成本优化](/glossary/cost-optimization)
