# 温度

## 概述

温度（Temperature）是控制大语言模型输出随机性的参数。较高的温度产生更多样化和创造性的输出，较低的温度产生更确定性和一致的输出。

## 为什么重要

- **创造性控制**：调节输出的创意程度
- **一致性控制**：控制输出的稳定性和可预测性
- **应用适配**：不同场景需要不同温度设置
- **调试工具**：帮助理解和调试模型行为

## 取值影响

- **0**：几乎确定的输出，选择最高概率结果
- **0.7**：平衡创造性和一致性
- **1.0**：原始概率分布
- **>1.0**：增加随机性，输出更不可预测

## 典型应用

- **创意写作**：温度 0.8-1.0
- **代码生成**：温度 0.2-0.4
- **问答系统**：温度 0.3-0.6
- **翻译任务**：温度 0.3 左右
