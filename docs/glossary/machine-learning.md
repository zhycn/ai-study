# 机器学习

## 概述

机器学习（Machine Learning）是 AI 的一个重要分支，是指让计算机系统从数据中自动学习和改进的算法和技术，无需显式编程。

## 为什么重要

- **数据驱动**：充分利用数据价值
- **自动进化**：从数据中持续学习
- **应用广泛**：解决各种实际问题
- **AI 基础**：现代 AI 的核心技术

## 学习范式

- **监督学习**：从标注数据学习
- **无监督学习**：从无标注数据学习
- **半监督学习**：部分标注数据学习
- **强化学习**：通过交互学习

## 核心算法

- **回归**：预测连续值
- **分类**：预测离散类别
- **聚类**：发现数据模式
- **降维**：减少数据维度

## 与其他概念的关系

- 是 [深度学习](/glossary/deep-learning) 的基础
- 包括 [神经网络](/glossary/neural-network) 算法
- 与 [机器学习](/glossary/machine-learning) 相互促进
