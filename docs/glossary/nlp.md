# 自然语言处理

## 概述

自然语言处理（Natural Language Processing，NLP）是 AI 的重要分支，致力于让计算机理解、解释和生成人类语言。

## 为什么重要

- **人机交互**：实现自然语言交流
- **信息获取**：从文本中提取信息
- **自动化**：文本任务自动化
- **多语言**：支持全球语言

## 核心任务

- **文本分类**：情感分析、主题分类
- **命名实体识别**：提取实体信息
- **机器翻译**：跨语言转换
- **问答系统**：自动问答
- **文本生成**：内容创作

## 技术发展

- **规则时代**：基于规则的方法
- **统计时代**：统计学习方法
- **深度学习时代**：神经网络方法
- **大模型时代**：预训练+微调

## 与其他概念的关系

- 是 [大语言模型](/glossary/llm) 的上游学科
- 使用 [Embedding](/glossary/embedding) 表示文本
- 受益于 [Transformer](/glossary/transformer) 突破
