# 函数调用

## 概述

函数调用（Function Calling）是让大语言模型能够调用外部函数或 API 的技术，使模型能够执行实际操作而不仅仅是生成文本。

## 为什么重要

- **能力扩展**：突破模型知识限制
- **实时信息**：获取最新信息
- **执行行动**：执行实际操作
- **任务闭环**：完成端到端任务

## 工作原理

1. 定义函数签名和描述
2. 用户提出需要工具的请求
3. 模型识别需要调用函数
4. 模型生成函数参数
5. 执行外部函数调用
6. 将结果加入上下文
7. 模型基于结果生成回复

## 应用场景

- **数据库查询**：查询和操作数据库
- **API 调用**：调用外部服务 API
- **文件操作**：读写文件
- **计算执行**：执行数学计算
