# 闭源模型

## 概述

闭源模型是指由商业公司开发和维护，不公开模型权重和架构的 AI 模型，用户通过 API 调用使用。

## 为什么重要

- **商业化**：AI 公司的主要商业模式
- **质量保证**：专业团队持续优化
- **服务保障**：SLA 服务承诺
- **快速迭代**：持续推出新版本

## 代表产品

- **OpenAI**：GPT-4、GPT-4o
- **Anthropic**：Claude 3.5
- **Google**：Gemini Pro
- **Cohere**：企业 AI 平台

## 商业模式

- **按量计费**：Token 用量计费
- **订阅制**：固定费用订阅
- **企业版**：定制化企业服务
- **增值服务**：额外功能和服务

## 与开源对比

| 维度 | 闭源 | 开源 |
|------|------|------|
| 易用性 | 高 | 中 |
| 定制性 | 低 | 高 |
| 成本 | API 费用 | 运维成本 |
| 隐私 | 数据需上传 | 可本地部署 |

## 与其他概念的关系

- 通过 [API](/glossary/api) 提供服务
