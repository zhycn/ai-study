# 开源模型

## 概述

开源模型是指权重和架构对公众开放的 AI 模型，用户可以自由使用、修改和部署。

## 为什么重要

- **可定制**：根据需求修改模型
- **透明性**：可审查模型行为
- **成本低**：无需 API 调用费用
- **社区支持**：活跃的社区生态

## 主流开源模型

- **Llama**：Meta 开源大模型
- **Qwen**：阿里开源大模型
- **DeepSeek**：深度求索开源模型
- **Mistral**：法国 AI 公司开源

## 开源协议

- **Apache 2.0**：宽松开源协议
- **GPL**：传染性开源协议
- **自定义协议**：如 Llama 协议
- **CC**：知识共享协议

## 与闭源对比

| 维度 | 开源 | 闭源 |
|------|------|------|
| 定制性 | 高 | 低 |
| 透明度 | 高 | 低 |
| 成本 | 运维成本 | API 调用 |
| 支持 | 社区 | 官方 |
