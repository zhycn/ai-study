# 对话系统

## 概述

对话系统（Conversational AI）是指能够与人类进行自然语言交互的 AI 系统，包括聊天机器人、虚拟助手等，是 AI 技术最广泛的应用之一。

## 为什么重要

- **自然交互**：最直观的人机交互方式
- **应用广泛**：客服、教育、娱乐等多场景
- **商业价值**：降低人工成本，提升服务效率
- **技术验证**：是 AI 能力的综合体现

## 技术架构

- **理解层**：意图识别和实体提取
- **对话管理**：状态追踪和策略选择
- **生成层**：回复生成和表达
- **知识层**：知识检索和增强

## 关键能力

- **多轮对话**：保持上下文连续性
- **意图理解**：准确理解用户意图
- **个性化和风格**：适应不同场景
- **多语言支持**：跨语言对话能力

## 与其他概念的关系

- 依赖 [大语言模型](/glossary/llm) 提供对话能力
- 使用 [RAG](/glossary/rag) 接入知识库
