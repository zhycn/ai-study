# Agent 编排

## 概述

Agent 编排（Agent Orchestration）是指协调和管理多个 AI Agent 的工作流程，确保它们按正确的顺序、条件和方式协作完成任务的技术。

## 为什么重要

- **任务分解**：将复杂任务合理分配给不同 Agent
- **流程控制**：管理 Agent 之间的执行顺序和依赖关系
- **状态管理**：跟踪和管理跨 Agent 的上下文状态
- **错误处理**：处理单个 Agent 失败的情况，保证整体稳定性

## 核心能力

- **工作流引擎**：定义和执行 Agent 协作流程
- **消息队列**：Agent 之间的异步通信
- **状态机**：管理任务状态转换
- **监控面板**：实时查看 Agent 运行状态

- 与 [工作流](/glossary/workflow) 概念紧密相关
