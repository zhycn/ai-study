# 提示词工程

## 概述

提示词工程（Prompt Engineering）是一门设计和优化提示词（Prompt）的学科，通过精心构造的输入文本，引导大语言模型（LLM）输出符合预期的结果。

## 为什么重要

提示词是与大模型交互的最直接方式。掌握提示词工程可以：

- 显著提升模型输出的质量和准确性
- 减少幻觉（Hallucination）和错误回答
- 解锁模型的隐藏能力，实现复杂任务

## 核心技巧

- **明确指令**：清晰描述任务目标
- **提供上下文**：给予足够的背景信息
- **示例引导**：使用 Few-shot 示例示范期望输出
- **角色设定**：为模型指定专业角色
- **格式约束**：规定输出的结构和格式

## 与其他概念的关系

- 建立在 [AI](/glossary/ai) 和大语言模型技术之上
- 与 [RAG](/rag/) 结合可实现知识增强型问答
- 是构建 [Agent](/agent/) 的基础能力之一
