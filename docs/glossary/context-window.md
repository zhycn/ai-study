# 上下文窗口

## 概述

上下文窗口（Context Window）是指语言模型一次能处理的最大 [Token](/glossary/token) 数量，定义了模型在单次生成时能考虑的输入范围。

## 为什么重要

- **记忆容量**：决定了模型能记住多少上下文信息
- **任务范围**：影响能处理的任务复杂度
- **成本控制**：更长的上下文意味着更高的成本
- **技术挑战**：扩展上下文窗口是重要研究方向

## 扩展技术

- **滑动窗口**：固定窗口滑动处理长文本
- **上下文扩展**：通过微调扩展上下文长度
- **稀疏注意力**：降低长上下文的计算成本
- **分层记忆**：区分短期和长期上下文

## 与其他概念的关系

- 限制可输入的 [Token](/glossary/token) 数量
- 影响 [RAG](/glossary/rag) 的分块策略
- 与 [记忆](/glossary/memory) 概念相关
