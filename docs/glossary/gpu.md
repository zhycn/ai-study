# GPU

## 概述

GPU（Graphics Processing Unit，图形处理器）最初为图形渲染设计，因其大规模并行计算能力现已成为 AI 模型训练和推理的核心硬件。

## 为什么重要

- **并行计算**：数千个核心同时计算
- **深度学习**：神经网络训练的必要硬件
- **推理加速**：生产环境推理加速
- **成本效率**：相比 CPU 更高算力性价比

## GPU 在 AI 中的作用

- **模型训练**：提供大规模并行计算能力
- **推理服务**：加速模型推理响应
- **数据处理**：加速数据预处理
- **分布式训练**：多卡并行训练大模型

## 主流产品

- **NVIDIA A100**：数据中心级 GPU
- **NVIDIA H100**：最新一代 AI 加速卡
- **NVIDIA RTX 系列**：消费级 GPU
- **AMD Instinct**：AMD AI 加速卡
