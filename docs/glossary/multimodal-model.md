# 多模态模型

## 概述

多模态模型是指能够同时处理和理解多种模态（如文本、图像、音频、视频）信息的 AI 模型，是 AI 发展的重要方向。

## 为什么重要

- **更自然**：符合人类感知世界的方式
- **更强大**：多模态信息互补
- **更通用**：跨模态理解和生成
- **应用广**：自动驾驶、医疗等

## 模态类型

- **文本-图像**：图文理解和生成
- **文本-语音**：语音对话
- **文本-视频**：视频理解和生成
- **全部模态**：统一多模态

## 技术路线

- **对齐派**：分别编码后对齐
- **融合派**：早期融合多模态
- **原生派**：统一多模态训练
- **自回归**：多模态生成

## 代表模型

- **GPT-4V**：图像理解
- **Gemini**：原生多模态
- **Sora**：视频生成
- **GPT-4o**：实时语音视频

## 与其他概念的关系

- 基于 [Transformer](/glossary/transformer) 架构
- 是 [生成式 AI](/glossary/generative-ai) 的重要组成
- 使用 [Embedding](/glossary/embedding) 实现跨模态
