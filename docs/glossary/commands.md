# Commands

## 概述

Commands（命令）是指用户与 AI Agent 交互时使用的指令格式，通过特定的命令语法，用户可以触发 Agent 执行特定操作或调用特定技能。

## 为什么重要

- **明确意图**：清晰的命令格式帮助 Agent 理解用户需求
- **标准化交互**：统一的命令格式提高交互效率
- **功能入口**：是用户调用 Agent 各种能力的入口
- **可扩展性**：易于添加新命令来扩展 Agent 功能

## 命令类型

- **系统命令**：控制 Agent 行为的命令
- **技能命令**：调用特定技能的命令
- **查询命令**：获取信息或状态
- **操作命令**：执行具体操作

- 是用户与 [Agent](/glossary/agent) 交互的接口
- 触发相应的 [Skills](/glossary/skills) 执行
