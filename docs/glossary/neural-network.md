# 神经网络

## 概述

神经网络（Neural Network）是受人脑神经元结构启发而设计的计算模型，是深度学习和机器学习的核心算法基础。

## 为什么重要

- **通用近似**：能够近似任意函数
- **自动学习**：从数据中自动学习
- **并行计算**：适合大规模并行处理
- **多任务**：可处理多种类型任务

## 网络结构

- **输入层**：接收输入数据
- **隐藏层**：进行特征提取和转换
- **输出层**：产生最终输出
- **激活函数**：引入非线性

## 经典架构

- **MLP**：多层感知机
- **CNN**：卷积神经网络
- **RNN**：循环神经网络
- **Transformer**：注意力机制网络

## 训练方法

- **反向传播**：计算梯度
- **梯度下降**：更新参数
- **优化器**：Adam、SGD 等
- **正则化**：防止过拟合
