# 深度学习

## 概述

深度学习（Deep Learning）是机器学习的一个分支，使用具有多个隐藏层的深层神经网络来学习数据的表征，是当前 AI 发展的核心技术。

## 为什么重要

- **性能突破**：在多个任务上超越传统方法
- **自动特征**：无需人工设计特征
- ** Scaling 规律**：模型越大性能越强
- **广泛应用**：图像、NLP、语音等领域

## 核心架构

- **全连接网络**：基础网络结构
- **卷积神经网络（CNN）**：图像处理
- **循环神经网络（RNN）**：序列处理
- **Transformer**：当前主流架构

## 关键要素

- **海量数据**：训练深度模型的基础
- **强大算力**：需要 GPU/TPU 支持
- **模型规模**：参数数量持续增长
- **训练技巧**：优化器、正则化等
