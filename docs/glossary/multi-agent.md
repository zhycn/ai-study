# 多 Agent 系统

## 概述

多 Agent 系统（Multi-Agent System）是指由多个独立的 AI Agent 组成的系统，这些 Agent 可以相互通信、协作或竞争，共同完成单个 Agent 无法处理的复杂任务。

## 为什么重要

- **分工协作**：将复杂任务分解给不同 Agent，专注各自领域
- **并行处理**：多个 Agent 同时工作，提高效率
- **冗余容错**：某个 Agent 失败不影响整体任务完成
- ** emergent behavior**：群体协作产生单个 Agent 不具备的能力

## 核心模式

- **协作式**：Agent 共享目标，协同工作
- **竞争式**：Agent 各自目标不同，通过竞争优化结果
- **分层式**：上层 Agent 协调下层 Agent，形成层级结构
