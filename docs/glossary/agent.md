# Agent

## 概述

Agent（智能体）是指能够感知环境、做出决策并执行行动的 AI 系统，具有自主性、反应性和目标导向性。

## 为什么重要

- **自动化**：从被动响应到主动执行
- **复杂性**：处理多步骤复杂任务
- **自主性**：减少人工干预
- **落地应用**：AI 实际应用的主要形态

## 核心能力

- **感知**：理解环境和用户输入
- **规划**：分解任务并制定计划
- **行动**：执行具体操作
- **工具使用**：调用外部工具扩展能力
- **记忆**：保持上下文和经验

## Agent 架构

- **LLM**：核心推理引擎
- **Tools**：外部工具能力
- **Memory**：记忆机制
- **Planning**：规划能力
- **Reflection**：反思和自改进
