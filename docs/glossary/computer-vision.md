# 计算机视觉

## 概述

计算机视觉（Computer Vision，CV）是 AI 的重要分支，致力于让计算机理解和处理图像和视频内容。

## 为什么重要

- **视觉智能**：机器的"眼睛"
- **应用广泛**：安防、医疗、自动驾驶
- **多模态**：与语言模型结合
- **自动化**：图像任务自动化

## 核心任务

- **图像分类**：识别图像内容
- **目标检测**：定位和识别对象
- **图像分割**：像素级分类
- **图像生成**：AI 生成图像
- **视频分析**：理解视频内容

## 技术发展

- **传统方法**：特征提取+分类器
- **CNN 时代**：卷积神经网络
- **Transformer 时代**：Vision Transformer
- **多模态时代**：视觉语言模型

## 与其他概念的关系

- 是 [多模态模型](/glossary/multimodal-model) 的基础之一
- 使用 [深度学习](/glossary/deep-learning) 方法
- 与 [图像生成](/glossary/image-generation) 紧密相关
