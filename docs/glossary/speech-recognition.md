# 语音识别

## 概述

语音识别（Speech Recognition）是指将人类语音转换为文本的技术，也称为自动语音识别（ASR）。

## 为什么重要

- **输入便捷**：比打字更快捷的输入方式
- **无障碍**：帮助听障人士获取信息
- **应用广泛**：语音输入、字幕等
- **多模态交互**：语音交互的基础

## 技术架构

- **声学模型**：将声音转换为音素
- **语言模型**：将音素转换为文本
- **端到端模型**：直接从声音到文本
- **大模型融合**：结合 LLM 提升准确率

## 评估指标

- **词错误率（WER）**：识别错误的词比例
- **字符错误率（CER）**：识别错误的字符比例
- **实时率（RTF）**：处理速度与实时比例
- **方言支持**：不同口音的识别能力

## 应用场景

- **语音输入**：手机、电脑语音输入
- **会议转录**：会议内容自动记录
- **字幕生成**：视频自动加字幕
- **语音命令**：控制智能设备
