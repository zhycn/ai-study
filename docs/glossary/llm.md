# 大语言模型

## 概述

大语言模型（Large Language Model，LLM）是指基于海量文本训练、拥有数十亿参数的语言模型，能够理解和生成人类语言。

## 为什么重要

- **通用智能**：展现类似人类的语言能力
- **广泛应用**：对话、写作、编程等
- **技术突破**：AI 发展的重要里程碑
- **新范式**：Prompt as Code

## 核心特征

- **大规模**：数十亿参数
- **海量训练**：万亿 Token 训练
- **涌现能力**：规模产生新能力
- **通用性**：多任务通用

## 发展历程

- **GPT-1**：2018年，首个 GPT
- **GPT-3**：2020年，175B 参数
- **ChatGPT**：2022年，对话产品
- **GPT-4**：2023年，多模态能力
