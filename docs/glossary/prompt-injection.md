# 提示词注入

## 概述

提示词注入（Prompt Injection）是一种攻击技术，通过在用户输入中注入恶意指令来操控 AI 模型的行为，使其绕过安全限制或执行非预期操作。

## 为什么重要

- **安全风险**：最常见的大模型安全威胁
- **隐蔽性强**：攻击往往不易察觉
- **危害大**：可能导致敏感数据泄露
- **防护难**：完全防护具有挑战性

## 攻击类型

- **直接注入**：直接覆盖系统提示
- **间接注入**：通过上下文注入
- **嵌套注入**：多重嵌套攻击
- **软注入**：通过角色扮演绕过

## 防护措施

- **输入过滤**：过滤可疑输入
- **输出验证**：验证模型输出
- **指令分离**：明确区分指令和用户输入
- **持续更新**：根据新攻击持续改进
